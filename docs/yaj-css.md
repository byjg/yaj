# yo(selector).css()

## yo(selector).css(style) 

Get the style value of the selector

```javascript
var value = yo('.myclass').css('font-family');
```

## yo(selector).css(style, value) 

Set the style value into the selector;

```javascript
yo('.myclass').css('font-family', 'Arial');
```
